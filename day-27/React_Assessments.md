# Student Assessments (React)

## 1. Temperature Converter

Create a React app that: - Takes a temperature input. - Has radio
buttons to select conversion: Celsius to Fahrenheit or Fahrenheit to
Celsius. - Auto-updates result on any change. - Displays valid converted
output using formulas: - C → F: F = (C × 9/5) + 32 - F → C: C = (F --
32) × 5/9

## 2. Prime Number Detector

Create a React app that: - Accepts a number input. - Automatically
detects if it is prime. - Displays appropriate messages: - "X is
prime" - "X is not prime" - Handles invalid values gracefully. - Uses an
efficient prime-check method (up to sqrt(n)).

## 3. BMI Calculator

Create a React app that: - Takes height (cm) and weight (kg). -
Automatically calculates BMI = weight / (height/100)\^2. - Displays BMI
category: - Underweight - Normal - Overweight - Obese

## 4. Fixed Deposit Calculator

Create a React app that: - Accepts principal amount, interest rate, and
number of years. - Automatically calculates maturity amount using: - A =
P × (1 + r/100)\^t - Displays maturity amount clearly.
